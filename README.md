## 주요 학습 내용 (Main Concepts)

- **파이썬 기초**: 변수, 조건문, 함수, 반복문 등
- **데이터 구조**: 리스트, 딕셔너리, 세트, 튜플 등의 활용
- **NumPy**: 수치 연산과 배열 다루기
- **SymPy**: 심볼릭 수학 계산
- **웹 크롤링**: Selenium을 활용한 자동화 및 데이터 수집
- **데이터 분석**: Pandas와 Matplotlib를 사용한 데이터 처리 및 시각화
- **Flask**: 웹 애플리케이션 개발 실습
